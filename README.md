# 🎲 Random Number Guessing Game

A fun, interactive terminal-based guessing game built using Python. The program generates a secret number, and the player tries to guess it with helpful hints along the way!

## ✨ Features
* **Random Generation**: Uses Python's `random` module to pick a secret number every match.
* **Smart Feedback**: Tells the player if their guess is "Too high" or "Too low" to guide them to the correct answer.
* **Score tracking**: Tracks the number of attempts it took the player to find the correct number.

## 🚀 How to Run the Project
1. Open your terminal in the project folder.
2. Run the command:
   ```bash
   python random_num_guess_game.py
   ```
